# Verwendete npm-Pakete

## Projekt

Super Shop 

## Verwendete Pakete

### Next.js

Verwendung:

* Routing zwischen Seiten
* Server Components
* Rendering der Anwendung

Beispiel:

```tsx
import Link from "next/link";
```

---

### React

Verwendung:

* Komponenten
* State Management
* Hooks

Beispiel:

```tsx
import { useState } from "react";
```

---

### Tailwind CSS

Verwendung:

* Styling der Benutzeroberfläche
* Responsive Design

Beispiel:

```tsx
className="bg-black text-white px-4 py-2"
```

---

### Lucide React

Verwendung:

* Icons für Navigation und Benutzeroberfläche

Verwendete Icons:

* Heart
* ShoppingBag
* User
* Search
* Menu

Beispiel:

```tsx
import { Heart } from "lucide-react";
```

## Fazit

Im Projekt werden mehrere externe npm-Pakete verwendet. Besonders Next.js, React, Tailwind CSS und Lucide React bilden die technische Grundlage der Anwendung.
